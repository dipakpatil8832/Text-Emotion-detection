## 🎭 Text Emotion Detection using Machine Learning & NLP

🚀 An end-to-end NLP project that identifies human emotions from text using Machine Learning.  
This system analyzes user-written text and accurately classifies it into emotions such as:  

😊 Happy | 😢 Sad | 😠 Angry | 😨 Fear | 😲 Surprise | 😐 Neutral  

It is highly useful for sentiment analysis, chatbots, social media monitoring, customer feedback analysis, and mental health insights.  

🧠 Project Overview

Understanding emotions hidden in text is a key challenge in Natural Language Processing (NLP).
This project applies text preprocessing, feature extraction, and machine learning models to detect emotions from raw text data.

✔ Cleaned and preprocessed text  
✔ Extracted meaningful features using TF-IDF  
✔ Trained ML model for multi-class emotion classification   
✔ Deployed with a Streamlit web app for real-time predictions  

🚀 Features

🔤 Text Preprocessing (tokenization, stopwords, stemming)

🧠 ML Model for Emotion Classification

📊 TF-IDF Vectorization

🎯 High Accuracy on Cleaned Dataset

📈 Performance Metrics & Visualization

🖥️ Simple Streamlit App for Live Prediction
