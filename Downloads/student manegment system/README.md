🎓 Student Management System (Python)

A simple command-line based Student Management System built using Python.
This project allows users to add, view, search, update, and delete student records efficiently.

🚀 Features
➕ Add new student details
📋 View all students in tabular format
🔍 Search student by roll number
✏️ Update student marks
❌ Delete student record
🔁 Menu-driven program (easy to use)
🛠️ Tech Stack
Language: Python 🐍
Library: Pandas (for tabular display)
